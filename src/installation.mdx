# Installation

Frontier is a React form library for GraphQL applications.

In order to use Frontier, you will need:

- `react` (`^16.8.6`)
- `apollo-client` (`^2.5.1`)
- `graphql-tag` (`^2.10.1`)

<br />

Then, install `frontier-forms`:

```
yarn add frontier-forms
```

*OR*

```
npm i frontier-forms
```


You now can [get started](/getting-started) with Frontier.
